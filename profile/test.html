직무능력사항

          펼쳐보기          
           
             
HTML5 / CSS3
             
                웹 표준과 접근성을 준수한 시맨틱 마크업으로 구조를 설계하고,                 CSS(Flexbox, Grid 등)를 활용하여                 반응형 레이아웃 및 스타일 구현이 가능합니다.                 유지보수를 고려한 모듈화된 CSS 작성을 지향합니다.              
           
           
             
JavaScript (ES6+)
             
                JavaScript를 활용한                 동적 DOM 조작 및 이벤트 처리를 이해하고 활용할 수                   있습니다.                 fetch API와 async/await를 사용한                 비동기 통신(AJAX) 및 JSON 데이터 처리 경험이                 풍부하며, 재사용성을 고려한 코드 작성을 지향합니다.              
           
           
             
JQuery
             
                jQuery를 이용한 다양한 동적 UI 기능 및 이벤트 구현 경험이                 있습니다.              
           
           
             
MySQL, ORACLE, Supbase
             
                데이터베이스 설계/구축 및 SQL 활용이 가능하며, 이를 바탕으로                 Java(백엔드)와 JavaScript(프론트엔드)를 연동하여 CRUD 기능 웹                 페이지 제작이 가능합니다.              
           
           
             
Git / GitHub
             
                Git/GitHub를 활용한 버전 관리 및 팀 협업 경험이 있습니다.                 https://github.com/rlaskarb              
           
           
             
React
             
                React를 사용하여 SPA(Single Page Application) 구조의 맥주 생산                 ERP 프론트엔드를 개발한 경험이 있습니다. 컴포넌트 기반으로 UI를                 구축하고, useState/useEffect 등을 활용한 상태 관리 및                 fetch/axios를 통한 API 연동을 경험했습니다.              
           
           
             
Figma
             
                디자인 시안 및 협업을 위한 Figma 활용에 많이 사용하며, 이를                 바탕으로 UI 구현이 가능합니다              
           
           
             
Adobe Photoshop
             
                웹 페이지에 필요한 기본적인 이미지 편집(크기 조절, 보정) 및 배경                 제거 작업이 가능합니다.              
           
           
             
Backend(Java, Spring, RESTAPI)
             
                풀스택 교육을 통해 Java, Spring Boot 기반의 기본적인 REST API                 설계 및 개발 경험을 쌓았습니다. 이를 통해 프론트엔드와 백엔드 간                 데이터 연동 과정을 이해하고 있어, 원활한 협업 및 효율적인 기능                 구현에 기여할 수 있습니다.              
           
           
             
Postman
             
                개발 및 테스트 과정에서 Postman을 활용하여 REST API 명세를                 확인하고 통신 테스트를 진행한 경험이 있습니다.              
           
           
             
MS Office
             
업무를 위한 word, excel, ppt 업무가 가능합니다.
           
           
             
BootStrap
             
                JavaScript 반응형 프레임워크 중 하나인 bootstrap을 사용하여                 태그형 CSS를 사용할 수 있고, 해상도에 따른 최적화된 반응형                 웹사이트의 구축이 가능합니다.
